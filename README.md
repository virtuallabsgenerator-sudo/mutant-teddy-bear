# mutant-teddy-bear
Mutant Teddy Bear est un projet de jeu univers immersif mêlant cartes à collectionner, univers narratif riche et éléments interactifs. Le projet vise à créer un écosystème centralisé où chaque élément est généré ou enrichi par intelligence artificielle (IA), tout en garantissant une création originale et innovante.
